FlexEmbed

CSS for responsive, intrinsic ratio embeds. Includes modifier classes for 3:1, 2:1, 16:9, and 4:3 aspect ratios.

<div class="FlexEmbed">
  <div class="FlexEmbed-ratio [modifier class]"></div>
  <div class="FlexEmbed-content">
    <img alt="" height="1000" src="http://www.gifpng.com/1000/aaaaaa?text=+" width="1000" />
  </div>
</div>
